# Quick Start

This guide takes you from a blank project to a full AI-generated track in under five minutes.

## 1. Open the AI Panel

Click the **⚡ AI** button in the main toolbar, or press **Ctrl+Shift+A**.

The AI Panel docks as a floating MDI sub-window inside the workspace.

## 2. Describe Your Music

In the **Generate** tab, type a description in the prompt field:

```
Upbeat lo-fi hip hop with jazz chords, a smooth bass line, and vinyl crackle
```

Optionally set:

- **Model** — ACE-Step 1.5 (quality) or DiffRhythm 2 (speed)
- **Genre**, **Key**, **Tempo**, **Duration**

## 3. Generate

Click **⚡ Generate** to create one track, or **⚡×3 Best of 3** to generate three
variations side-by-side and pick your favorite.

Generation takes ~15–30 seconds on a mid-range GPU (longer on CPU).

## 4. The Track Appears in Your Project

The generated audio clip is automatically inserted as a new **SampleTrack** in the
Song Editor, ready to play, loop, and edit.

## 5. Split Stems (optional)

Right-click any audio clip in the Song Editor → **Wavy Labs AI → Split Stems…**

Each stem (vocals, drums, bass, other) is inserted as a separate color-coded SampleTrack:

| Stem | Color |
|------|-------|
| Vocals | Red |
| Drums | Orange |
| Bass | Blue |
| Guitar | Green |
| Piano | Purple |
| Other | Grey |

## 6. Export

Use **File → Export → WAV/MP3/FLAC** as you would in any LMMS project.
