│ Wavy Labs — AI DAW Product Plan                                                                                                                                                       │
│                                                                                                                                                                                       │
│ Context                                                                                                                                                                               │
│                                                                                                                                                                                       │
│ Wavy Labs is a new AI-powered Digital Audio Workstation forked from LMMS (GPL-2.0, C++/Qt6).                                                                                          │
│ It targets all skill levels with a tiered UX, operates on a freemium+subscription model,                                                                                              │
│ runs all AI 100% locally, and ships V1 with: Music Generation, Stem Splitting, Vocal                                                                                                  │
│ Generation, AI Mixing/Mastering, Prompt Commands (AI that controls LMMS features via                                                                                                  │
│ natural language), and Code-to-Music (generate music from code/data).                                                                                                                 │
│                                                                                                                                                                                       │
│ ---                                                                                                                                                                                   │
│ Product Vision                                                                                                                                                                        │
│                                                                                                                                                                                       │
│ "The DAW that listens to you."                                                                                                                                                        │
│                                                                                                                                                                                       │
│ Wavy Labs layers AI features on top of a full-featured desktop DAW without replacing                                                                                                  │
│ the traditional workflow. Users start simple (prompt → generate → done) and grow into                                                                                                 │
│ advanced production. All AI runs locally — no data sent to servers, no ongoing API cost.                                                                                              │

│ Feature Specifications                                                                                                                                                                │
│                                                                                                                                                                                       │
│ 1. AI Generation Panel                                                                                                                                                                │
│                                                                                                                                                                                       │
│ - MDI SubWindow in LMMS workspace (toggle via toolbar button)                                                                                                                         │
│ - Text prompt field → generate → audio clip inserted as new SampleTrack                                                                                                               │
│ - Model selector (ACE-Step / DiffRhythm)                                                                                                                                              │
│ - Genre, tempo, key, duration controls                                                                                                                                                │
│ - "Continue from selection" mode (extend existing clip)                                                                                                                               │
│ - Generation history with preview player                                                                                                                                              │
│ - Free tier: counter showing remaining daily generations                                                                                                                              │
│                                                                                                                                                                                       │
│ 2. Stem Splitter                                                                                                                                                                      │
│                                                                                                                                                                                       │
│ - Right-click any SampleTrack clip → "Split Stems"                                                                                                                                    │
│ - OR dedicated Stem Splitter panel                                                                                                                                                    │
│ - Select 2, 4, or 6 stems (Pro: 6 stems = vocals, drums, bass, piano, guitar, other)                                                                                                  │
│ - Output: each stem becomes a new SampleTrack, color-coded                                                                                                                            │
│ - Progress bar with estimated time                                                                                                                                                    │
│ - Uses Demucs v4 htdemucs_ft model                                                                                                                                                    │
│                                                                                                                                                                                       │
│ 3. Vocal Generation                                                                                                                                                                   │
│                                                                                                                                                                                       │
│ - In the AI Panel, switch to "Vocal" mode                                                                                                                                             │
│ - Input: lyrics text + voice style selector                                                                                                                                           │
│ - Output: .wav file inserted as SampleTrack                                                                                                                                           │
│ - Uses Bark (MIT) with voice presets                                                                                                                                                  │
│ - Pro feature                                                                                                                                                                         │
│                                                                                                                                                                                       │
│ 4. AI Mixing / Mastering                                                                                                                                                              │
│                                                                                                                                                                                       │
│ - Right-click Mixer channel → "AI Mix Assist"                                                                                                                                         │
│ - OR "Master" button in Mixer view                                                                                                                                                    │
│ - Analyzes all tracks, suggests EQ/compression/reverb settings                                                                                                                        │
│ - "Reference track" mode: match the sonic profile of an uploaded reference                                                                                                            │
│ - Applies settings as LMMS AutomationPatterns (editable, not destructive)                                                                                                             │
│ - Pro feature                                                                                                                                                                         │
│                                                                                                                                                                                       │
│ 5. Prompt Commands (AI that controls LMMS)                                                                                                                                            │
│                                                                                                                                                                                       │
│ Natural language interface that controls the DAW:                                                                                                                                     │
│                                                                                                                                                                                       │
│ Example commands:                                                                                                                                                                     │
│ - "Add a 4-bar drum loop at 120 BPM"                                                                                                                                                  │
│ - "Transpose the selected clip up 3 semitones"                                                                                                                                        │
│ - "Set the reverb on channel 3 to 40%"                                                                                                                                                │
│ - "Create a verse-chorus-verse arrangement"                                                                                                                                           │
│ - "Duplicate track 2 and pan it hard right"                                                                                                                                           │
│                                                                                                                                                                                       │
│ Architecture:                                                                                                                                                                         │
│ - Prompt bar (always visible, Cmd+K shortcut)                                                                                                                                         │
│ - Mistral 7B parses intent → structured JSON action                                                                                                                                   │
│ - C++ action dispatcher maps JSON → LMMS Engine API calls                                                                                                                             │
│ - Action schema covers: track management, clip editing, mixer controls, piano roll, automation                                                                                        │
│ - Studio tier feature                                                                                                                                                                 │
│                                                                                                                                                                                       │
│ 6. Code to Music                                                                                                                                                                      │
│                                                                                                                                                                                       │
│ Convert code or structured data into music:                                                                                                                                           │
│                                                                                                                                                                                       │
│ Input modes:                                                                                                                                                                          │
│ - Python/JS snippet → parse data structures/loops → map to musical parameters                                                                                                         │
│ - CSV/JSON data → sonification (map columns to pitch, velocity, tempo)                                                                                                                │
│ - MIDI-like DSL → custom text format for describing music programmatically                                                                                                            │
│                                                                                                                                                                                       │
│ Example:                                                                                                                                                                              │
│ # User writes this in Wavy Labs Code Editor                                                                                                                                           │
│ track("drums").pattern([1,0,0,1,0,0,1,0], bpm=140)                                                                                                                                    │
│ track("bass").melody([C3, E3, G3, C4], duration="quarter")                                                                                                                            │
│ track("synth").generate("ambient pad", key="C minor")                                                                                                                                 │
│                                                                                                                                                                                       │
│ Architecture:                                                                                                                                                                         │
│ - Monaco code editor embedded in Qt (via QWebEngineView)                                                                                                                              │
│ - Python interpreter in AI backend parses DSL                                                                                                                                         │
│ - Outputs MIDI + audio clips back to LMMS tracks                                                                                                                                      │
│ - Studio tier feature                                                                                                                                                                 │
│                                                                                                                                                                                       │
